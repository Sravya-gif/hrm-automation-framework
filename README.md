# HRM Automation Framework

## Project Overview

This project is a Selenium-based automation framework developed for testing an HR Management System.

## Tech Stack

* Java
* Selenium WebDriver
* TestNG
* Maven

## Project Structure

* src → Test scripts & Page Object Model
* screenshots → Execution screenshots
* pom.xml → Dependencies
* testng.xml → Test configuration

## Features

* Login Automation
* Employee Management Testing
* Screenshot on Failure
* Reusable Page Object Model

##  How to Run

1. Import as Maven Project in Eclipse
2. Run testng.xml
